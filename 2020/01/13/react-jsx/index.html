什么是jsx，什么是React元素

  • jsx只是个语法糖，最终会被编译成React.createElement，
  • ReactElement是构建React应用的最小单位
  • React元素用来描述你界面上看到的内容，
  • React元素其实就是个简单的js对象：{type:’h1’,props:{children: “hello”}}

元素的更新：默认React元素是不可以更新的，(freeze)默认不可变，如果界面需要修改，那我们就需要创建一个新元素，来渲染新元素：


//react的元素默认是不可修改的，如果需要修改，需要每次创建一个新的element，

setInterval(() => {
    let element = <div>{new Date().toLocaleString()}</div>    //定时器每次都会生成一个新的element

    /*
    Object.freeze(element.pros);   冻结对象的属性也不可修改：
    Object.freeze(element);   属性本身不允许修改
    */

    ReactDom.render(element, document.getElementById('root'));

}, 1000);
