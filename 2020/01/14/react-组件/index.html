react-组件

发表于 2020-01-14 | 分类于 react

如何定义组件，两种方式 1.函数式 2.对象式 返回是一个React顶级的元素，

函数式

函数中：react如何定义并渲染页面的 ？

  • 把所有的属性组合成一个对象（props） –所有属性都可以在该属性对象中访问到，函数是不会实例化的，一锤子买卖，用完即销毁，不消耗性能
  • 把属性对象作为参数传给函数组件
  • 函数组件会返回一个React元素，
  • 然后由ReactDom.render方法来吧虚拟dom（React元素）转为真实元素，并且插入到页面中。

对象方式(注意函数和对象方式的相互关系)

类中如何渲染类组件 ？

  • 收集属性组成props对象，
    • 把props对象传递给Welcome类的构造函数，new Welcome(props)，然后获取到Welcome类的实例； （创建许多实例会消耗性能）
    • 调用实例的render方法进行渲染，获得返回的React元素。
    • 把此元素渲染到页面上。

函数式编程：什么叫纯函数

  • 在函数体内部不能改变传入的参数
  • 相同的参数一定要返回相同的值，
  • 不能修改作用域变量之外的值：
